Se remoja el pan con el vino. Se pica la cebolla, el tomate y las hierbas, muy menudos. Se majan las especias. Se mezcla todo con el pan deshecho y los huevos batidos. Se calienta una cucharadita de aceite en una sartén pequeña. Se forman las tortillas, con una cucharada de mezcla. Se escurren sobre papel absorbente. Para ganar tiempo, se pueden hacer en dos sartenes a la vez.
