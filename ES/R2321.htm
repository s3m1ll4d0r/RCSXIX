   PARA 25 UNIDADES

100 g de almendras molidas

100 g de almendras machacadas gruesas

1 Kg de azúcar molido

3 claras

Aceite para untar las bandejas

Se baten las claras ligeramente. Se mezclan las almendras con el azúcar. Se añaden las claras a cucharadas, hasta obtener una pasta que se pueda manejar con las manos. Se calienta el horno. Se preparan las bandejas de horno con papel untado de aceite. Se forman bolitas de pasta y se colocan en las bandejas, separadas entre sí. Se aplastan ligeramente. Se introducen en el horno, a temperatura suave, durante 35 minutos. Se retiran de las bandejas cuando estén fríos.
